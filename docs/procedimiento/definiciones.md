# 3. Definiciones y Abreviaturas

## 3.1 Definiciones

| Término | Definición |
|---|---|
| Requerimiento | Necesidad funcional o no funcional solicitada por el área usuaria. |
| Product Owner | Responsable funcional del requerimiento o sistema (representa al usuario). |
| Historia de Usuario | Descripción ágil de una funcionalidad desde la perspectiva del usuario. |
| Control de Cambios | Proceso formal para gestionar modificaciones de alcance, cronograma o entregables. |
| Pruebas Funcionales | Validación del correcto funcionamiento del sistema frente a lo solicitado. |
| Paso a Producción | Autorización formal para liberar un sistema o funcionalidad a ambiente productivo. |
